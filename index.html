Estrutura básica do HTML

A estrutura básica de uma página HTML consiste nas seguintes Tags:

  • HTML - indica ao navegador que tudo que se encontra desta Tag trata-se de um Html
  • head - elementos que irão carregar antes que a página seja exibida ao usuário
  • Dentro da tag head há também a tag title, essa tag define o título da página
  • body - tag que sinaliza o conteúdo que irá aparecer para o usuário
(Voltar)

Tags

Entre as tags principais existem diversas outras, como por exemplo:

  • i - deixa o texto em itálico
  • strong - deixa o texto em negrito

É possível usar uma tag dentro de outra pra dar mais de um efeito, por exemplo, usar a tag strong dentro da tag i para deiaxar um texto em negrito e itálico

A maioria das tags tem uma abertura e um fechamento usando barra /html, mas existem algumas exceções como por exemplo:

  • input - cria um campo de texto para usuário preencher
  • img - insere imagem dento da página
  • hr - cria uma linha para dividir conteúdo
(Voltar)

Atributos das tags

Atributos são as propriedades que as tags possam ter, algumas de uso geral, outras, específicas

  • id - identifica um elemento
  • style - específico do CSS pode estilizar um elemento dento do html
  • class - usado para padronização de um elemento definido pelo CSS
    • type - específico da tag input, define o uso do campo criado
    • src - específico da tag img, indica caminho da imagem a ser inserida no site
(Voltar)

Textos

Existem diversas tags relacionadas à tipografia, as principais são:

  • h1 até h6 - define o título da página, sendo o título principal definido com a tag h1, e os subtítulos recebendo h2 à h6
  • p - tag que define um parágrafo
  • strong - coloca texto em negrito
  • i - coloca texto em itálico
  • u - coloca uma linha embaixo do texto
  • mark - destaca texto como se fosse uma caneta marca texto
  • sup - coloca texto em sobrescrito, como por exemplo o símbolo de marca registrada R
(Voltar)

Lista ordenada e lista não ordenada

  • ol - tag que define uma lista ordenada
  • ul - tag que define uma lista não ordenada

    • As listas não recebem texto comum dentro delas, elas necessitam de uma tag filha, a tag li, que define os itens da lista

    As listas também são usadas para criar um menu, sendo posteriormente configuradas no CSS

(Voltar)

Links

Os links são responsáveis pela navegação entre páginas, outros sites, ou até memo entre itens dentro de uma mesma página. Usamos a tag a para definir um link, ela tem um atributo específico e obrigatório, o href, que define o link a ser usado na tag

Outros atributos que podem ser usados dentro da tag a são:

  • o target, que define que o link deve ser aberto em uma nova aba, através da opção blank
  • o title, que mostra uma legendalegenda ao colocar o cursor sobre o link, a qual pode ajudar na navegação
(Voltar)

Outras tags:

  • small - cria um texto pequeno
  • del - intervalo de texto excluído do documento
  • abbr - mostra significado de uma abreviação ao passar cursor do mouse sobre a palavra como por exemplo html
  • blockquote -
    "marca texto como uma citação"

    • Existem outras tags, com por exemplo font, que no html5 deixou de ser usada, substituída pela configuração no CSS
